# LOLA

Проект: сначала сайт, затем приложение.

## Структура

- **Корень** — сайт (статический):
  - `index.html` — главная страница
  - `css/style.css` — стили
  - `js/main.js` — скрипты
- **app/** — заготовка под приложение (позже)

## Как открыть сайт локально

1. Откройте `index.html` в браузере (двойной клик),  
   **или**
2. Запустите простой сервер из папки LOLA, например:
   ```bash
   npx serve .
   ```
   или
   ```bash
   python -m http.server 8000
   ```
   затем откройте в браузере: http://localhost:8000

## Дальше

- Дорабатывать страницы и стили в корне — это и есть сайт.
- Когда будете готовы к приложению — добавляйте проект в папку `app/`.
