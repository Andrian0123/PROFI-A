## Осталось одно действие (только вы можете его сделать)

Включить GitHub Pages в настройках репозитория может только владелец аккаунта (вход в GitHub). Без этого шага сайт не откроется по ссылке.

### Как включить (один раз)

**Вариант 1 — через файл в проекте**  
Дважды щёлкните по файлу **`ВКЛЮЧИТЬ_GITHUB_PAGES.bat`** в корне проекта. Откроется страница настроек GitHub. Войдите в аккаунт, если попросят, затем в блоке **Build and deployment** → **Source** выберите **GitHub Actions**.

**Вариант 2 — вручную**  
1. Откройте: https://github.com/Andrian0123/PROFI-A  
2. Вкладка **Settings** → слева **Pages**  
3. **Source** → **GitHub Actions**

После этого:
- Вкладка **Actions** покажет запущенный workflow (или он запустится при следующем push).
- Через 1–2 минуты сайт будет доступен: **https://andrian0123.github.io/PROFI-A/**
