# LOLA 2.0 — Definition of Done (MVP)

## Общие критерии
- Задача покрыта тестом (unit/integration/e2e по месту).
- Документация обновлена.
- Нет blocker-багов.
- Логи и ошибки читаемы.

## Чат
- Сообщения отправляются/получаются стабильно.
- Ошибки сети и лимита обрабатываются.
- Сессия сохраняется и корректно восстанавливается.

## Микрофон (STT)
- Разрешение микрофона запрашивается корректно.
- Речь конвертируется в текст и может быть отредактирована перед отправкой.
- Ошибки распознавания имеют понятный UX.

## Ответ текст/аудио (TTS)
- Переключатель режима работает и сохраняется.
- В аудио-режиме есть воспроизведение/пауза и прогресс.
- Если TTS недоступен, есть fallback на текст без крэша.

## Подписка / Paywall
- Ограничения free применяются корректно.
- Покупка и восстановление подписки работают.
- Статус подписки синхронизирован с backend.
