# LOLA 2.0 — Privacy Policy (Draft)

## 1. Какие данные собираем
- Данные аккаунта: email, идентификатор пользователя.
- Контент диалогов: текст запросов и ответов.
- Технические данные: логи ошибок, версия приложения, тип устройства.
- Подписка: статус покупки и срок действия.

## 2. Для чего используем
- Предоставление функций приложения (чат, персонажи, история).
- Улучшение качества сервиса и диагностика ошибок.
- Верификация подписок.

## 3. Голосовые данные
- Голос используется только для распознавания речи (STT) и/или озвучки ответа (TTS).
- Если используется облачный STT/TTS, данные обрабатываются внешним провайдером по его политике.

## 4. Хранение и защита
- Передача данных по HTTPS.
- Доступ к данным ограничен ролями.
- Секреты хранятся в защищённом виде.

## 5. Права пользователя
- Запрос на удаление аккаунта и данных.
- Запрос на экспорт доступных данных.

## 6. Контакты
- Указать email поддержки и юридическое лицо перед публикацией.
