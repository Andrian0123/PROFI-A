## 2. Метаданные видео (разрешение, FPS, длительность)

Сейчас в окружении нет **OpenCV** и **ffprobe**, поэтому автоматически получены только путь и размер файла.

Чтобы получить в отчёте:
- разрешение (ширина × высота),
- частоту кадров (FPS),
- количество кадров и длительность в секундах,
- кодек,

установите один из вариантов:

```bash
pip install opencv-python-headless
```

или установите **ffmpeg** и добавьте его в PATH.

После установки выполните:

```bash
python tools/analyze_scan_video.py E:\PROFI-A\document_5242313571123105378.mp4
```

Скрипт выведет полные метаданные и при необходимости сможет извлечь кадры для пайплайна 3D-сканера.

## 3. Использование в пайплайне сканера

Этот файл можно использовать как источник кадров для сервера сканирования PROFI-A:

1. **Извлечь кадры** (после установки OpenCV):
   ```bash
   python tools/analyze_scan_video.py E:\PROFI-A\document_5242313571123105378.mp4 --extract E:\PROFI-A\videos\frames --every 5 --max-frames 30
   ```

2. **Отправить кадры на сервер** через API `POST /api/v1/scan/process` (как при сканировании с телефона) или подключить вызов к скрипту/приложению.

3. Сервер вернёт размеры помещения (по потолку), откосы и короба при наличии распознанных плоскостей.
