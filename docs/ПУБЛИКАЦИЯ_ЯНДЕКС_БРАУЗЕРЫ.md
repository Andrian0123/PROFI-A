# Публикация ПРОФЙ-А в экосистеме Яндекса и для пользователей Яндекс Браузера

Дата: 2026 г.

---

## 1. Страница приложения (сайт)

Страница в стиле приложения: **`site/profi-a.html`**.

- Открывается в **Яндекс Браузере** без ограничений (обычный HTML/CSS).
- Рекомендуется разместить по постоянному URL (например `https://ваш-домен.ru/profi-a.html` или на поддомене).
- После размещения добавьте в [Яндекс.Вебмастер](https://webmaster.yandex.ru/) свой сайт и при необходимости укажите мета-тег проверки (см. ниже в коде страницы).

---

## 2. Магазины приложений для пользователей с Яндекс ID

Отдельного магазина приложений «Яндекс» для Android сейчас нет (ранее был закрыт). Пользователи, которые заходят через Яндекс ID, чаще всего ставят приложения из:

- **RuStore** — вход через Яндекс ID поддерживается. Публикация ПРОФЙ-А в RuStore даёт доступ аудитории в т.ч. тем, кто использует Яндекс Браузер и сервисы Яндекса.

### Публикация в RuStore (кратко)

1. Регистрация: [RuStore для разработчиков](https://www.rustore.ru/developer) (вход через VK ID, почту, телефон, **Яндекс** или Госуслуги).
2. Создать приложение, загрузить AAB/APK (подписанный).
3. Заполнить карточку: название «ПРОФЙ-А», описание (можно взять из `docs/ТЕКСТЫ_ДЛЯ_МАГАЗИНА_И_САЙТА.md`), скриншоты, ссылки на политику конфиденциальности и пользовательское соглашение (URL страницы, где они опубликованы, например страница приложения на сайте).
4. Отправить на модерацию (сроки см. в консоли RuStore).

В карточке приложения укажите ссылку на страницу приложения (где лежит `profi-a.html`), чтобы пользователи из Яндекс Браузера могли перейти на описание и при необходимости скачать из RuStore.

---

## 3. Проверка и индексация в Яндексе

- **Яндекс Браузер** — страница открывается как обычный сайт, дополнительных настроек не требуется.
- **Поиск Яндекса** — после публикации добавьте сайт в [Яндекс.Вебмастер](https://webmaster.yandex.ru/), при необходимости вставьте в `<head>` страницы мета-тег проверки прав (см. комментарий в `site/profi-a.html`).
- В карточке приложения в RuStore и на странице сайта используйте одни и те же формулировки про «вспомогательный инструмент» и «ориентировочный характер» расчётов (как в `docs/ТЕКСТЫ_ДЛЯ_МАГАЗИНА_И_САЙТА.md`).

## 4. Что уже подготовлено в проекте

| Элемент | Где |
