# Сайт приложения ПРОФЙ-А

Все страницы оформлены в стиле приложения (палитра из Color.kt). Готово к проверке и публикации.

**Полная пошаговая инструкция:** откройте **[ПОШАГОВО.md](ПОШАГОВО.md)** — там все шаги по порядку (проверка, GitHub Pages, Tilda, свой хостинг).

## Файлы

| Файл | Назначение |
|------|------------|
| **index.html** | Главная страница: описание приложения, дисклеймер, возможности, кнопка «Скачать в Google Play», ссылки на политику и соглашение |
| **privacy.html** | Политика конфиденциальности (полный текст) |
| **terms.html** | Пользовательское соглашение (полный текст) |
| **profi-a.html** | Копия главной (можно использовать как отдельная страница, например на другом домене) |

## Как проверить локально

1. Откройте папку `site` в проводнике.
2. Дважды щёлкните по **index.html** — откроется в браузере.
3. Проверьте переходы: «Политика конфиденциальности», «Пользовательское соглашение», «На главную».

Или откройте в браузере по пути:  
`file:///E:/PROFI-A/site/index.html`

## Публикация на GitHub Pages (автоматически)

В проекте настроена автоматическая публикация: при push в ветку `main` сайт из папки `site/` выкладывается на GitHub Pages.

1. В репозитории на GitHub: **Settings** → **Pages** → **Source**: выберите **GitHub Actions**.
2. Отправьте изменения в репозиторий (push ветки `main`), например:
   ```text
   git add .
   git commit -m "Сайт ПРОФЙ-А"
   git push origin main
   ```
3. Дождитесь окончания workflow **Deploy site to GitHub Pages** (вкладка **Actions**).
4. Сайт будет доступен по адресу:  
   `https://<ваш-username>.github.io/PROFI-A/`  
   (имя репозитория может отличаться — смотрите в Settings → Pages).

После этого в карточке приложения укажите ссылки:
- Политика конфиденциальности: `https://<username>.github.io/PROFI-A/privacy.html`
- Пользовательское соглашение: `https://<username>.github.io/PROFI-A/terms.html`

## Публикация на свой хостинг

1. Загрузите на хостинг **все файлы** из папки `site/`:
   - index.html
   - privacy.html
   - terms.html
   - profi-a.html (по желанию)
2. Убедитесь, что главная страница сайта открывается по адресу вида `https://ваш-домен.ru/` или `https://ваш-домен.ru/index.html`.
3. В карточке приложения в Google Play и RuStore укажите:
   - Ссылку на политику конфиденциальности: `https://ваш-домен.ru/privacy.html`
   - Ссылку на пользовательское соглашение: `https://ваш-домен.ru/terms.html`
4. После публикации приложения в магазине замените в `index.html` и `profi-a.html` ссылку на кнопке «Скачать в Google Play» на реальную ссылку из магазина.
