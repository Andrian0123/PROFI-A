## Что нужно сделать вам (один раз)

### Включить GitHub Pages

1. Откройте в браузере: **https://github.com/Andrian0123/PROFI-A**
2. Вкладка **Settings** → слева **Pages**
3. В блоке **Build and deployment** → **Source** выберите **GitHub Actions**
4. Сохраните (если есть кнопка Save)

### Дождаться деплоя

1. Вкладка **Actions** — должен быть запущен workflow **Deploy site to GitHub Pages**
2. Дождитесь зелёной галочки (обычно 1–2 минуты)

### Открыть сайт

После успешного деплоя сайт будет доступен по адресу:

**https://andrian0123.github.io/PROFI-A/**

- Главная: https://andrian0123.github.io/PROFI-A/
- Политика конфиденциальности: https://andrian0123.github.io/PROFI-A/privacy.html
- Пользовательское соглашение: https://andrian0123.github.io/PROFI-A/terms.html

Эти ссылки можно указать в карточке приложения в Google Play и RuStore.

## Tilda и свой хостинг

- **Tilda:** откройте `site/tilda-block.html` в Блокноте → Ctrl+A → Ctrl+C → в Tilda создайте страницу → блок «HTML-код» → Ctrl+V → опубликовать. Подробно в **ПОШАГОВО.md** (способ Б).
- **Свой хостинг:** загрузите файлы из папки `site/` (или из архива `profi-a-site.zip`) в корень сайта. Подробно в **ПОШАГОВО.md** (способ В).
