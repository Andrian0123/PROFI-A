# Все задачи пошагово — статус

| № | Задача | Статус |
|---|--------|--------|
| 1 | Собрать APK | ⬜ **Выполните один раз** — запустите **ВСЕ_ЗАДАЧИ_ОДНИМ_ЗАПУСКОМ.bat** (или соберите в Android Studio, затем ДОПИСАТЬ_APK_И_ОТПРАВИТЬ.bat) |
| 2 | Скопировать APK в site/profi-a.apk | Входит в батник |
| 3 | Закоммитить и push на GitHub | ✅ Выполнено (скрипты и страницы уже на GitHub) |

---

## Один запуск — все шаги

Запустите **ВСЕ_ЗАДАЧИ_ОДНИМ_ЗАПУСКОМ.bat** в корне проекта (`E:\PROFI-A`):

1. Соберёт APK (используется Java из Android Studio).
2. Скопирует его в `site/profi-a.apk`.
3. Сделает commit и push на GitHub.

Через 1–2 минуты ссылка **https://andrian0123.github.io/PROFI-A/profi-a.apk** заработает.

**Важно:** закройте другие терминалы и вкладки с Gradle/сборкой, чтобы не было блокировки файлов.

---

## Если батник выдаёт ошибку сборки

1. Откройте проект в **Android Studio** → **Build → Build APK(s)**.
2. После сборки запустите **ДОПИСАТЬ_APK_И_ОТПРАВИТЬ.bat** — он положит APK на сайт и отправит на GitHub.
