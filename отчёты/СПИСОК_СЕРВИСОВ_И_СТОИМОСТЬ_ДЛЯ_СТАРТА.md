# Список сервисов и стоимость для старта PROFI-A

Ориентировочные затраты на запуск проекта. Цены примерные, зависят от региона и провайдера. Обновляйте перед принятием решений.

---

## 1. Обязательные для публикации приложения

| Сервис | Назначение | Стоимость | Период |
|--------|------------|-----------|--------|
| **Google Play Developer Account** | Публикация приложения в Google Play | **$25** | Единоразово |
| **Google Play Billing** | Приём оплаты подписок в приложении | **0%** (комиссия Google 15–30% с транзакции) | — |

---

## 2. Бэкенд (Auth, Support, Scan, Verification)

Варианты: один общий сервер или раздельные сервисы.

### Вариант A: Один VPS (всё на одном сервере)

| Сервис | Что разворачиваем | Примерная стоимость | Период |
|--------|-------------------|----------------------|--------|
| **VPS (Linux)** | Node или другой runtime для Auth + Support + Verification; Python (uvicorn) для Scan | **$5–15 / мес** (например, 1–2 vCPU, 2–4 GB RAM) | В месяц |
| **Домен** (опц.) | api.yourapp.com | **$10–15 / год** | В год |
| **SSL** | HTTPS (Let's Encrypt бесплатно) | **$0** | — |

**Итого вариант A:** от **~$5/мес** без домена до **~$15/мес** с запасом по ресурсам.

### Вариант B: Бессерверные / managed-сервисы

| Сервис | Провайдер (примеры) | Примерная стоимость | Период |
|--------|----------------------|----------------------|--------|
| **Auth + API** | Firebase, Supabase, AWS API Gateway + Lambda | **$0–25 / мес** (free tier часто хватает на старт) | В месяц |
| **Scan (Python)** | AWS Lambda + Container / Railway / Render | **$0–15 / мес** (зависит от объёма запросов) | В месяц |
| **Support (тикеты)** | Можно включить в тот же API или использовать внешний сервис (Zendesk, Freshdesk — есть free tier) | **$0–20 / мес** | В месяц |

**Итого вариант B:** **$0–60/мес** в зависимости от выбора и трафика.

### Вариант C: Только разработка / MVP без продакшен-бэкенда

| Сервис | Режим | Стоимость |
|--------|--------|-----------|
| **Auth / Support / Verification** | В приложении пустые URL → демо и mailto | **$0** |
| **Scan** | Свой ПК или один дешёвый VPS только для Scan | **$0** или **$3–5/мес** |

## 3. Хранение артефактов скана (меш, превью, JSON)

| Сервис | Назначение | Примерная стоимость | Период |
|--------|------------|----------------------|--------|
| **S3-совместимое хранилище** (AWS S3, Selectel, Yandex Object Storage, MinIO) | Файлы сканов по scan_id | **$0–5 / мес** (небольшие объёмы на старте) | В месяц |
| **CDN** (опц.) | Раздача файлов быстрее | Часто входит в тариф хранилища или **$0–10 / мес** | В месяц |

На старте можно не подключать и оставить заглушки URL — **$0**.

---

## 4. Верификация покупок (подписка)

| Сервис | Назначение | Стоимость |
|--------|------------|-----------|
| **Сервер с endpoint POST /verify** | Уже входит в VPS или serverless из п. 2 | — |
| **Google Play Developer API** | Использование API для проверки подписок | **$0** (доступ в рамках аккаунта разработчика $25) |

Отдельной платы за API нет; нужен только аккаунт разработчика.

---

## 5. Опционально: ИИ и улучшения

| Сервис | Назначение | Примерная стоимость | Период |
|--------|------------|----------------------|--------|
| **GPU-сервер** (если своя модель для скана/документа) | Обучение или инференс тяжёлых моделей | **$20–100+ / мес** | В месяц |
| **Managed ML** (Google Cloud AI, AWS SageMaker и т.п.) | Обучение/инференс без своего железа | По тарифам провайдера, часто **$0** на free tier для тестов | — |
| **scikit-learn / лёгкие модели** | Уже работают на CPU на том же VPS, что и Scan | **$0** (учтено в VPS) | — |

На старте можно обойтись без отдельного ML-хостинга — **$0**.

## 6. Сводка: минимальный старт

| Стадия | Что включаем | Ориентир стоимости |
|--------|----------------------|---------------------|
| **Только публикация в Google Play** | Аккаунт разработчика $25 | **$25** единоразово |
| **Старт с одним сервером** | Google Play + VPS (Auth, Support, Scan, Verify на одном хосте) | **$25** разово + **$5–15/мес** |
| **Старт с доменом** | То же + домен | **$25** разово + **~$10–15/год** + **$5–15/мес** |
| **Старт без своего бэкенда** | Только Google Play, в приложении демо (пустые URL) | **$25** единоразово |

---

## 7. Рекомендация для старта

- **Минимум:** Google Play Developer **$25** один раз; бэкенды не обязательны (демо и mailto в приложении).
- **Оптимально для запуска с реальными пользователями:** **$25** разово + **один VPS $5–10/мес** (Node + Python на одном сервере, Auth, Support, Scan, Verify), домен по желанию.
- Хранилище артефактов и отдельный ML-хостинг можно добавить позже при росте нагрузки.

*Цены актуальны на момент составления; проверяйте на сайтах провайдеров.*
