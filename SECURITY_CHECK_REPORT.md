# Отчёт проверки проекта на вирусы и взлом

**Дата проверки:** 2026-03-01  
**Папка:** PROFI-A (исходный код приложения)

---

## Результаты проверки кода

### Вредоносный код — **не обнаружен**

- **Выполнение кода:** в коде приложения (`app/`) нет вызовов `Runtime.getRuntime().exec()`, `ProcessBuilder`, `eval()`, `ScriptEngine`, загрузки нативных библиотек через `System.load` из произвольных путей.
- **Шифрование/обфускация:** нет подозрительного использования `Base64.decode` для исполняемого контента, `Cipher` с захардкоженными ключами и т.п.
- **Скрипты:** проверка паттернов в `.bat`/`.ps1` не выявила вызовов `Invoke-Expression`, `-EncodedCommand`, загрузки и запуска исполняемых файлов извне.

### Безопасность и утечки данных

- **Пароли и ключи:** в репозитории нет захардкоженных паролей, API-ключей или токенов. Упоминания "password"/"token" относятся к строкам интерфейса и нормальной логике авторизации (поля ввода, запросы к серверу).
- **Сеть:** в `network_security_config.xml` разрешён HTTP только для локальных адресов (10.0.2.2, localhost, 127.0.0.1, типичные домашние 192.168.x.x) — для разработки и локального backend это ожидаемо. Публичные домены не добавлены в cleartext.
- **Манифест:** запрошены только `INTERNET` и `CAMERA` (камера не обязательна). Нет лишних или подозрительных разрешений (SMS, контакты, микрофон без необходимости и т.д.).

### Рекомендации

1. **Антивирус:** запустите полное сканирование папки проекта штатным антивирусом (Windows Defender или другим):
   - ПКМ по папке `E:\PROFI-A` → «Сканировать с помощью Microsoft Defender» (или ваш AV).
2. **Секреты:** не добавляйте в репозиторий файлы `.env`, `local.properties` с ключами, пароли в коде. `local.properties` уже в `.gitignore`.
3. **Скрипты:** перед запуском неизвестных `.bat`/`.ps1` из корня проекта просматривайте их содержимое — в этой проверке подозрительных вызовов не найдено, но при появлении новых скриптов их стоит проверять.

---

**Итог:** по статическому анализу кода признаков вирусов и взлома не выявлено. Для полной уверенности выполните проверку папки антивирусом на вашем ПК.
