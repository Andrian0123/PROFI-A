========================================
  ПРОФИ-А — загрузка на Google Drive
========================================

Приложение: ПРОФИ-А (PROFI-A)
Версия: 1.0.0 (debug-сборка для установки по ссылке)
Назначение: составление смет, актов КС-2/КС-3, расчёты по комнатам, справочники материалов и видов работ, сканер помещения (3D) и сканер документов.

Требования: Android 8.0 (API 26) и выше.

В этой папке: файл PROFI-A-1.0.0-debug.apk (или с другой версией в имени).

----------------------------------------
ШАГИ ДЛЯ ЗАГРУЗКИ НА ДИСК
----------------------------------------
1. Зайдите на drive.google.com
2. Создайте папку (например «ПРОФИ-А — для тестировщиков»)
3. Загрузите в неё файл APK из этой папки
4. По файлу: правая кнопка → Настройки доступа (или «Поделиться»)
5. Выберите «Доступ по ссылке» → «Все, у кого есть ссылка» (просмотр/скачивание)
6. Скопируйте ссылку и отправьте её получателям

----------------------------------------
ДЛЯ ПОЛУЧАТЕЛЕЙ (кратко)
----------------------------------------
• Скачать APK по вашей ссылке с Google Drive
• Разрешить установку из неизвестных источников (для браузера или приложения «Файлы»: Настройки → Безопасность / Приложения)
• Установить и запустить ПРОФИ-А

При первом запуске можно войти в аккаунт или использовать демо-режим. Сканер помещения — в форме комнаты (Фото 3D → Создать). Сканер документов — в меню: Документы → Сканер документов. До 01.06.2026 сканеры в этой сборке доступны для теста без подписки.
